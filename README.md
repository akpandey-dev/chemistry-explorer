# Chemistry Explorer

An interactive, JSON-powered periodic table and chemistry exploration system.
The project visualizes chemical elements in a dynamic periodic table and allows users to explore detailed information through multiple search modes. It is designed for students, developers, and science enthusiasts who want a lightweight, browser-based way to explore chemical data.
The dataset includes up to **120 elements**, each described using **36 detailed properties**, including both standard and extended entries.

Built using **HTML, CSS, JavaScript, and Python**, with multiple implementation formats to balance portability, modularity, and real-world development practices.

⚠️ **Disclaimer**  
Some element data is AI-generated and may contain inaccuracies. This project is intended for educational and experimental purposes only. Do not treat the data as authoritative for scientific,professional, or academic work.

---

## Features

- Lightweight web-based build
- Contains a dynamic periodic table
- Key Legend given for guidance
- Search option for elements

## Usage

No installation required.

1. Clone the repository(or download the specific file(s))
2. Open a version inside `versions/`
3. Choose **single-file** or **modular**
4. Open the HTML file in any modern browser
5. Start using
